The data contains four ID variables and 79 measurement variables. The four ID variables are as follows:
1 subject #subject # from 1:30
2 group #either "test" or "training"
3 activity #numerical code for activity, 1:6
4 activity_label #character label for activity 
  (1: WALKING, 2: WALKING_UPSTAIRS, 3: WALKING_DOWNSTAIRS, 4: SITTING, 5: STANDING, 6: LYING)

The 79 measurement variables represent either individual measures or averages of measures by subject
and activity in the all_data_mean_sd and sub_act_means tables, respectively. All measurements are
normalized on a scale of [-1,1]. More details can be found in the README file found in the data directory
